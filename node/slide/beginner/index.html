WEBとは

インターネット上で標準的に用いられている、文書の公開・閲覧システム

文字や画像、動画などを一体化したHTMLをネット上で公開し、閲覧することができる

補足

“Web”とは蜘蛛の巣の意味

大規模な文書間の繋がりを図示すると複雑な蜘蛛の巣のように見えることからこのように呼ばれる

また、Webは、Webページとも呼ばれ、HTMLやCSSなどの組み合わせで作られたページのことを指す
HTMLでは見出しや段落などの文字、表、別のページ飛ぶためのリンクなど記載し、CSSでは要素の大きさや配置、色などの見栄えを整える

複数の相互に関連する一連のページの集合体をサイト（Webサイト）という

HTMLとは

ざっくり言うと、文書（テキストファイル）のこと

Hyper Text Markup Language（ハイパーテキスト・マークアップ・ランゲージ）の略。

Webページの骨組みとなるもの。

※マークアップとは印をつけるの意。

wikipediaから抜粋した文を例にします

文章を打ったテキストファイル（拡張子が.txtのファイル）を作成します

ブラウザにドラッグアンドドロップしてみましょう

こんな感じで表示されます

ではファイルの拡張子を.txt→.htmlに変更して再度ブラウザで開いてみます

改行などはありませんが、ちゃんと表示されます

以上のことから、htmlもテキストファイルの一種だということがわかります

そこで必要なのが、タグというものでテキストを囲う、「マークアップ」という作業です

htmlは、ブラウザで正しく表示するために、適切なタグでマークアップをする必要があります。

h1 → 一番大きい見出し

h2 → 二番大きい見出し

h3 → 三番大きい見出し

p → 文章の段落

ul、li → リスト

また、文字だけでなく、画像や動画などを表示する機能も備えています。

CSSとは

HTML（骨組み）を装飾（肉付け）するもの

正式名称はCascading Style Sheets（カスケーディング・スタイル・シート）と呼ばれる。ウェブページのスタイルを指定するための言語。

CSSは、HTMLと組み合わせて使用する言語。 HTMLがウェブページ内の各要素の意味や情報構造を定義するのに対して、CSSではそれらをどのように装飾するかを指定する。

肉付けする

HTML内にプログラムを埋め込むことで、Webページに様々な動きや機能を付加し、HTMLやCSSでは表現できないものを作ることができる

Web製作の仕事とは

営業

製作の仕事を取ってくる職業

クライアントとの折衝、収支の管理など

製作現場に用件を伝える役割を担うこともあり、どれだけの情報を、クライアントの意向と相違なく吸い取り、クリエイティブチームに対して同じレベルで共有することができるかが重要

Webディレクター

製作現場を管理する職業

タスク管理、時間管理、クライアントとの折衝など多岐に渡ります。
場所によっては営業と兼務のところも

Webデザイナー

デザインを作る職業

主にPhotoshoやIllastratorを使用し、クライアントの意向に沿ったデザインを提案・作成する
場所によってはコーディングまでおこなうところも

フロントエンドエンジニア（コーダー）

サイトの「見える部分」部分を組み上げる職業

デザイナーが作ったものを元に、HTML、CSS、javascriptを書いてサイトを組み上げる。この一連の流れを「コーディング」と呼ぶ

バックエンドエンジニア（サーバーサイドエンジニア）

サイトの「見えない」部分を組み上げる職業

ユーザーの操作に応じて、動的に処理をおこなうためのプログラミングをする

例：検索窓で何かテキストを打ち込んで「検索」を押した際、そのテキストに応じた検索結果を表示させる

Webサイトを製作する上で、他にも様々な職業があります。
